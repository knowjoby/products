# Products Portfolio

A minimalistic portfolio showcasing all my GitHub projects.

## Features

- ✨ Clean, minimal design
- 🔍 Real-time search by name and description
- 🏷️ Filter by programming language
- 📊 Automatic updates from GitHub API
- 📱 Fully responsive

## How it works

The portfolio fetches your public GitHub repositories via the GitHub API and displays them in a beautiful, searchable grid. No authentication required for public repos.

## View Online

Visit: https://knowjoby.github.io/products/

## Local Development

Just open `index.html` in a browser. The page fetches live data from GitHub's public API.

## Customization

Edit `index.html` to:
- Change featured repos: Update the `FEATURED_REPOS` array
- Modify language colors: Update the `LANGUAGE_COLORS` object
- Change GitHub username: Update `GITHUB_USER` constant
